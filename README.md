# 理气论分析器 (Li-Qi Analyzer)

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Version](https://img.shields.io/badge/version-0.2.0-blue.svg)](./CHANGELOG.md)

一个用于分析宋明理学理气论或天道论究竟属于**理本论**还是**气本论或气学**的 AI Skill / 智能体工具。

## 简介

本 Skill 基于**刘昊**提出的经典诠释分析方法，通过识别文本对《易传》两个核心命题的处理方式，判断理气论或天道论的思想归属。

### 核心分析机制

所有宋明理学的理气论建构，皆围绕以下两句《易传》命题展开：

- **命题A**："形而上者谓之道，形而下者谓之器"
- **命题B**："一阴一阳之谓道"

### 两种诠释路径

| 路径 | 前提 | 核心立场 | 代表学者 |
|------|------|----------|----------|
| **朱子学路径（理本论）** | 以A为前提，协调B | 太极/理是形而上之道，阴阳是形而下之器 | 程颐、朱熹、薛瑄、胡居仁 |
| **气本路径（气学）** | 以B为前提，代入A | 一阴一阳即道，形而上者即气 | 张载、陆九渊、王廷相、吴廷翰、戴震、王夫之 |

## 安装方法

### 通用方法（适用于支持 Skill 的 AI 平台）

   - **其他平台**: 请参考对应平台的 Skill / 插件安装文档

### Git Clone

```bash
git clone https://github.com/liuhao1106/liqi-analyzer.git
```

然后将 `SKILL.md` 复制到你的 AI 平台 skills 目录。

## 使用方法

安装后，当你在支持 Skill 的 AI 平台中提供任何宋明理学相关文本（尤其是涉及理气论、天道论、太极阴阳等议题），系统会自动调用此 Skill 进行分析。

### 支持的文本类型

- 宋明理学家的原著摘录
- 关于理气论的论文段落
- 涉及"形而上者谓之道""一阴一阳之谓道""太极""阴阳""理气"等核心概念的文本

### 分析输出格式

Skill 会输出结构化的分析结果，包括：

1. **核心命题识别** - 文本中涉及的关键命题和概念
2. **前提立场** - 作者以命题A还是命题B为前提
3. **关键概念界定** - 形而上者、道、阴阳、太极等的界定
4. **诠释策略** - 如何处理两个命题之间的关系
5. **本原判断** - 宇宙生成的根本是理还是气
6. **综合判定** - 归属判定及理由

## 分析示例

### 示例1：朱熹《朱子语类》

> "一阴一阳之谓道"。阴阳是气，不是道，所以为阴阳者，乃道也。若只言"阴阳之谓道"，则阴阳是道。

**判定**：朱子学/理本论
- 以A为前提，采用添字诠释（"所以为阴阳者"）
- 理为形而上，气为形而下
- 理为气之本

### 示例2：王廷相

> 气乃理之本，理乃气之载。

**判定**：气本论/气学
- 以B为前提，气即道
- 反对添字诠释
- 气为理之本

### 示例3：吴廷翰（详论气本论）

> 何謂道？"一陰一陽之謂道"。何謂氣？一陰一陽之謂氣。然則陰陽何物乎？曰氣。然則何以謂道？曰：氣即道，道即氣。天地之初，一氣而已矣，非有所謂道者别爲一物，以並出乎其間也。氣之混淪，爲天地萬物之祖，至尊而無上，至極而無以加，則謂之太極。及其分也，輕清者敷施而發散，重濁者翕聚而凝結，故謂之陰陽。陰陽既分，兩儀、四象、五行、四時、萬化、萬事皆由此出，故謂之道。太極者，以此氣之極至而言也。陰陽者，以此氣之有動静而言也。道者，以此氣之爲天地人物所由以出而言也，非有二也。
>
> 曰：然則何以有理氣之别？曰：理也者，氣得其理之名，亦猶變易之謂易、不測之謂神之類，非氣之外别有理也。先儒以陰陽爲氣，以道爲理，是去"一陰一陽之謂道"之義而他求之過也。夫論道之書，以《易》爲宗，而言以孔子爲準，反而求之以吾心自信者爲實。今論道不本諸《易》而求言未合乎孔子，反覆之不得乎吾心，則雖先儒爲説，亦且自以爲未安，而又安可聽憑一切以爲然乎？

**判定**：气本论/气学
- 以B为前提，"氣即道，道即氣"，阴阳即气即道
- 太极非独立于气之物，而是"氣之極至"、"氣之混淪"
- 明确反对"先儒以陰陽爲氣，以道爲理"，指出这是"去'一陰一陽之謂道'之義而他求之過"
- 理只是"氣得其理之名"，非气外别有理
- 强调以《易》为宗、以孔子为准，反对离阴阳而言道

### 示例4：吴廷翰（批判朱子学添字诠释）

> "一陰一陽之謂道"。程子謂："離了陰陽便無道，所以陰陽是道也。"又曰："道非陰陽也，所以一陰一陽者道也。"朱子謂："一陰一陽之謂道，當離合看。"又曰："陰陽是氣不是道，所以爲陰陽者乃道也。"又曰："陰陽只是陰陽，道便是太極。"據是數説，雖不能離陰陽以言道，然其曰"所以爲陰陽"，終是有一物以爲陰陽先也。其曰"道便是太極"，"太極生陰陽"，終是有道而後有陰陽也。其曰"離了陰陽便無道"，其下以形影喻之，似又先有陰陽而後有道也。其曰"當離合看"，夫可離可合，終是道自道，陰陽自陰陽也。反覆求之，不惟義有未明，而其爲説亦自有不相合者。若其不立己見，只據聖人之言，以陰陽爲道，則太極、性命、理氣等名義，皆可一貫而無疑矣。

**判定**：气本论/气学
- 以B为前提，直接以"一陰一陽之謂道"为据，反对程朱的添字诠释（"所以爲陰陽"）
- 批判程朱"有一物以爲陰陽先"、"有道而後有陰陽"，指出其将道/太极置于阴阳之前
- 批判"當離合看"为"道自道，陰陽自陰陽"，割裂道与阴阳
- 主张"只據聖人之言，以陰陽爲道"，太极、性命、理气皆可一贯
- 明确反对理本论将形而上之道与形而下之阴阳割裂的诠释策略

### 示例5：湛若水（复杂案例）

> 《易》"一阴一阳"之训，即气即道也。气其器也，道其理也，天地之原也。

**判定**：修正朱子学
- 以B为前提，强调"即气即道"
- 但"合一之道主乎理"，根本立场仍以理为本
- 未动摇"理为气之本"的朱子学根基

## 适用学者

本 Skill 适用于分析以下学者的理气论述：

- **理本论**：朱熹、程颐、程颢、薛瑄、胡居仁、魏校
- **修正朱子学**：罗钦顺、湛若水
- **气本论**：张载、陆九渊、王廷相、吴廷翰、戴震、胡煦、王夫之

## 理论背景

本 Skill 的理论基础来自刘昊的两篇论文：

1. **《道兼理气：朱子学理气论的经典诠释背景及其影响》**，《哲学研究》2023年第7期 - 揭示朱子学理气论建构的经典诠释渊源及其后世影响
2. **《"形而上者"的双重视野：经典诠释下的理本与气本》**，《哲学门》第45期 - 探讨"形而上者"的双重视野，分析理本与气本两种宇宙观念

### 核心洞见

- 宋代以来的理气论建构，皆围绕"形而上者谓之道"和"一阴一阳之谓道"展开
- 判断的关键不在于是否使用"气"的概念，而在于**如何处理两个经典命题的关系**
- 判断修正派是否溢出朱子学框架，关键看：**在造化之本或生物之本的问题上，是否动摇了"理为气之本"的根本立场**

## 注意事项

1. **避免简单二元对立**：不要简单地将所有理气论分为"理学"和"气学"两派，注意中间立场和修正派
2. **关注自我定位**：学者的自我定位（是否自认朱子学继承者）可作为参考，但不应作为唯一标准
3. **区分不同层面**：注意区分"本原层面"（宇宙生成根源）和"构成层面"（现实存在结构）的不同论述
4. **警惕"先天/后天"分类的局限**：不要仅从人性论是否具有先天超越根据来分类，而应从宇宙观和天道观整体考察

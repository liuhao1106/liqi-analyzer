# 理气论分析器 (Li-Qi Analyzer)

基于刘昊提出的经典诠释分析方法，通过识别文本对《易传》两个核心命题的处理方式，判断理气论或天道论的思想归属。

## 核心分析机制

### 两个经典命题

所有宋明理学的理气论建构，皆围绕以下两句《易传》命题展开：

- **命题A**："形而上者谓之道，形而下者谓之器"
- **命题B**："一阴一阳之谓道"

### 两种诠释路径

#### 路径一：朱子学路径（理本论）

**前提**：以命题A为前提，试图协调命题B

**核心立场**：
- 太极/理是"形而上之道"
- 阴阳是"形而下之器"（即气）
- "一阴一阳之谓道" ≠ "阴阳即道"
- 采用添字诠释法："所以为阴阳者，乃道也"
- 道与阴阳"有分别而不相离"（不离不杂）

**逻辑结构**：
- 形而上者 = 道 = 理（无形而有理）
- 形而下者 = 器 = 气（有形有象）
- "一阴一阳"是气的运动，其"所以"循环者才是道
- 若只言"阴阳之谓道"，则阴阳是道（这是需要避免的结论）

**代表性论述**：
- 《朱子语类》卷七十四："阴阳是气，不是道，所以为阴阳者，乃道也。若只言'阴阳之谓道'，则阴阳是道。"
- 程颐："所以阴阳者道也"
- 强调"理在气先"（本原层面）与"理气不离"（构成层面）并存

#### 路径二：气本路径（气学/气本论）

**前提**：以命题B为前提，将B代入A

**核心立场**：
- "一阴一阳之谓道" = 气即道
- "形而上者"（无形者、不形者）即气
- 反对添字诠释，主张回到经典原意
- 道即气化流行

**逻辑结构**：
- 一阴一阳 = 道（命题B）
- 形而上者 = 道（命题A）
- ∴ 形而上者 = 一阴一阳 = 气
- 气是宇宙生成的本原，理是气之理/条理

**代表性论述**：
- 张载："一阴一阳不可以形器拘，故谓之道"
- 陆九渊："一阴一阳，已是形而上者，况太极乎？"
- 王廷相："气乃理之本，理乃气之载"
- 吴廷翰："以阴阳为道""以气为理"
- 戴震："道，犹行也；气化流行，生生不息，是故谓之道"
- 胡煦："孔子两言'道'，可以得出阴阳是形而上者"

### 关键区分点

判断一个文本属于哪种路径，核心在于识别：

1. **前提选择**：以A为前提还是以B为前提？
2. **对"一阴一阳"的理解**：是"气"还是"道"？
3. **对"形而上"的理解**：是"无形之理"还是"无形之气"？
4. **诠释方法**：是否采用"添字诠释"（如"所以阴阳者"）？
5. **本原判断**：宇宙生成的根本是理还是气？

### 中间立场与修正派

部分学者看似反对朱子学，但未必构成气本论：

- **理气不离派**（如薛瑄、部分明初朱子学者）：反对"理在气先"，但坚持"理为气之本"
- **理气一物派**（如罗钦顺）：强调理气统一，但未颠倒为"气在理先"
- **合一派**（如湛若水）：强调"理气一体""上下一体"，但根本立场仍以理为本

判断修正派是否溢出朱子学框架，关键看：**在造化之本或生物之本的问题上，是否动摇了"理为气之本"的根本立场**。

## 分析步骤

当用户提供需要分析的文本时，按以下步骤进行：

### 步骤1：提取核心命题
识别文本中是否涉及以下关键表述：
- "形而上者谓之道，形而下者谓之器"
- "一阴一阳之谓道"
- "太极"
- "阴阳"
- "理气"关系
- "道器"关系

### 步骤2：识别前提立场
判断作者以哪个命题为前提：
- 若以"形而上者谓之道"为前提，试图解释"一阴一阳之谓道" → 倾向朱子学路径
- 若以"一阴一阳之谓道"为前提，解释"形而上者谓之道" → 倾向气本路径

### 步骤3：分析关键概念界定
- "形而上者"被界定为什么？（理？气？道？）
- "阴阳"被界定为什么？（形而上？形而下？）
- "道"与"阴阳"的关系如何？（等同？根源？不离？）

### 步骤4：识别诠释策略
- 是否使用"添字诠释"（如"所以阴阳者""所以为阴阳者"）？
- 是否反对添字诠释、主张回到经典原意？
- 是否使用"不离不杂""有分别而不相离"等表述？

### 步骤5：判断本原归属
- 宇宙生成的本原/根源是理还是气？
- "理"与"气"何者为本？（理乃气之本？还是气乃理之本？）
- 是否主张"气载乎理，理出于气"或"理乃气之载"？

### 步骤6：综合判定
基于以上分析，给出判定：

- **朱子学/理本论**：以A为前提，理为形而上，气为形而下，采用添字诠释协调B，理为气之本
- **修正朱子学**：反对理在气先或理气二物，但仍坚持理为气之本，形而上者为理
- **气本论/气学**：以B为前提，气即道，形而上者即气，反对添字诠释，气为理之本
- **难以判定/中间立场**：文本信息不足，或存在内在矛盾，或属于合一论但本原归属不明确

## 注意事项

1. **避免简单二元对立**：不要简单地将所有理气论分为"理学"和"气学"两派，注意中间立场和修正派
2. **关注自我定位**：学者的自我定位（是否自认朱子学继承者）可作为参考，但不应作为唯一标准
3. **区分不同层面**：注意区分"本原层面"（宇宙生成根源）和"构成层面"（现实存在结构）的不同论述
4. **警惕"先天/后天"分类的局限**：不要仅从人性论是否具有先天超越根据来分类，而应从宇宙观和天道观整体考察
5. **重视经典诠释方法**：判断的关键不在于是否使用"气"的概念，而在于如何处理两个经典命题的关系
